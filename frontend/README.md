# Expression Tree Calculator Frontend

This is the frontend component of the Expression Tree Calculator project.

## Features
- Clean and responsive user interface
- Real-time calculation of mathematical expressions
- Visual feedback during calculations
- Support for standard mathematical operations (+, -, *, /, ^)
- Parentheses support for grouping operations

## How to Use
1. Enter a mathematical expression in the input field
2. Click "Calculate" or press Enter
3. View the result and postfix notation of the expression

## Supported Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Exponentiation (^)
- Parentheses for grouping ()

## Example Expressions
- `3 + 4 * 2`
- `(3 + 4) * 2`
- `3 + 4 * 2 / (1 - 5)^2`
- `2^3^2`

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
